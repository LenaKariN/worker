# Employees — CLI-приложение управления организационной структурой

> Индивидуальное задание на производственную практику.

Приложение позволяет создавать, изменять и просматривать сотрудников и должности 
организации **ООО «НЕДРА» (Nedra Digital)** — ИТ-компании, разрабатывающей инженерное ПО для нефтегазовой отрасли.

---

## Быстрый старт

```bash
make install   # установить проект и зависимости
make run       # запустить REPL
```

Приложение поддерживает два режима:
- **REPL** - `employees` (без аргументов): интерактивный диалог, данные живут между командами.
- **One-shot** - `employees employee list`: однократный запуск команды.

## Функции приложения

### Сотрудники (FR01–FR03)

| Команда                               | Действие                                                       |
|---------------------------------------|----------------------------------------------------------------|
| `employees employee create ...`       | Создать сотрудника (ФИО, телефон, зарплата, должность, период) |
| `employees employee list`             | Постраничный список всех сотрудников                           |
| `employees employee show --id N`      | Детальный просмотр сотрудника                                  |
| `employees employee update --id N`    | Диалоговое редактирование (пошаговый ввод полей)               |
| `employees employee delete --id N`    | Мягкое удаление                                                |
| `employees employee tenure --years N` | Сотрудники со стажем более N лет (с сортировкой)               |

### Справочник должностей (FR04)

| Команда                              | Действие                                            |
|--------------------------------------|-----------------------------------------------------|
| `employees position create ...`      | Создать должность (код, название, описание, период) |
| `employees position list`            | Постраничный список справочника                     |
| `employees position show --code X`   | Детальный просмотр должности                        |
| `employees position update --code X` | Диалоговое редактирование                           |
| `employees position delete --code X` | Мягкое удаление                                     |

Справка по любой команде: `employees --help`, `employees employee --help`.

## Как проверить (5 минут)

```bash
# 1. Создать справочник должностей
employees position create --code DEV --name "Разработчик"
employees position create --code LEAD --name "Ведущий разработчик"

# 2. Создать сотрудников
employees employee create --last-name Иванов --first-name Пётр --phone +79111234567 --salary 150000 --position DEV
employees employee create --last-name Петрова --first-name Анна --phone +79219876543 --salary 200000 --position LEAD --effective-from 2018-06-01

# 3. Список сотрудников
employees employee list

# 4. Детальный просмотр
employees employee show --id 1

# 5. Сотрудники со стажем > 5 лет
employees employee tenure --years 5

# 6. Диалоговое редактирование
employees employee update --id 1
#   -> ввести: 1 Сидоров -> Enter -> 0 -> Enter
employees employee show --id 1

# 7. Мягкое удаление
employees employee delete --id 1
employees employee list
```

Полный план тестирования: [`docs/Requirements/Аналитический_обзор_приложения_Employees.md`](docs/Requirements/Аналитический_обзор_приложения_Employees.md#5-тестирование).
